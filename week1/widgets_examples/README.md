# Widgets Examples (Semaine 1)

Bienvenue ! Ce projet est un **bac à sable de widgets** pour la semaine 1.  
L'objectif est simple : **comprendre la structure d'une app Flutter** et **jouer avec les widgets** sans se perdre dans le reste.

## À quoi vous devez faire attention

Dans ce projet, **concentrez-vous uniquement sur le dossier `lib/`**, surtout :
- `lib/widgets/` : les petits exemples de widgets (Row, Column, Card, Stack, etc.)
- `lib/screens/` : des écrans qui regroupent ces widgets
- `lib/previews/` : des prévisualisations rapides pour voir plusieurs widgets à la fois
- `lib/main.dart` : le point d'entrée de l'app (juste pour voir comment tout est lancé)

## Ce que vous pouvez ignorer pour l’instant

Vous verrez d'autres dossiers (android, ios, web, test, etc.).  
**Ne vous inquiétez pas : on n’y touche pas encore.**  
Ils sont là pour que Flutter puisse compiler l’app, mais **ce n’est pas le sujet de la semaine 1**.

## Comment naviguer rapidement

Si vous êtes un peu perdu, voici un chemin simple :
1) Ouvrez `lib/main.dart` pour comprendre le point de départ  
2) Passez par `lib/screens/` pour voir des exemples complets  
3) Explorez `lib/widgets/` pour apprendre chaque widget séparément  

L’idée est d’apprendre **par petits morceaux**, pas tout d’un coup.

Bon apprentissage !

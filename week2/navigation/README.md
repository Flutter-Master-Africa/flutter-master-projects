# Navigation (Semaine 2)

Ce projet sert à **démontrer la navigation** dans Flutter.

## Où regarder

Dans `lib/`, vous avez deux versions :
- `lib/navigation/01/` : navigation simple (push/pop sans données)
- `lib/navigation/02/` : navigation avec retour de résultat

Le reste du projet (android, ios, web, etc.) sert seulement à exécuter l’app sur un émulateur ou un téléphone physique.
